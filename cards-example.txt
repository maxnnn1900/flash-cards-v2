Q. Как создать переменную и вывести её значение в Python?
A. Используйте оператор присваивания и функцию print:
```python
x = 5
print(x)
```
---
Q. Как объявить функцию в Python?
A. Функция определяется с помощью ключевого слова def:
```python
def greet(name):
    return f"Hello, {name}!"
```
---
Q. Как выполнить цикл for по списку чисел?
A. Итерация по элементам списка выглядит так:
```python
numbers = [1, 2, 3]
for n in numbers:
    print(n)
```
---
Q. Чем отличается список от кортежа?
A. Список (list) изменяемый, кортеж (tuple) нет:
```python
my_list = [1, 2]
my_tuple = (1, 2)
```
---
Q. Как добавить элемент в конец списка?
A. Используйте метод append:
```python
items = []
items.append('a')
```
---
Q. Как создать словарь с ключом и значением?
A. Словарь задаётся в фигурных скобках:
```python
user = {'name': 'Alice', 'age': 30}
```

Q. Как проверить условие и вывести сообщение?
A. Используйте оператор if:
```python
x = 10
if x > 5:
    print('x больше 5')
```
---
Q. Как обработать исключение?
A. Используйте конструкцию try/except:
```python
try:
    value = int('a')
except ValueError:
    print('Ошибка преобразования')
```
---
Q. Как считать все строки из файла?
A. Откройте файл и используйте read:
```python
with open('data.txt') as f:
    content = f.read()
```

Q. Как создать генератор списка квадратов чисел?
A. Используйте list comprehension:
```python
squares = [n * n for n in range(5)]
```

A. Импорт выполняется через ключевое слово import:

```python
import math
```


Q. Как объединить два списка?
A. Можно использовать оператор +:

Q. Как быстро создать множество из списка?
A. Передайте список в функцию set:
```python
nums = [1, 2, 2, 3]
unique = set(nums)
```

A. Через модуль datetime:

```python
from datetime import datetime

```

Q. Как получить элементы списка кроме первого?
A. Сделайте срез со смещением 1:
```python
values = [1, 2, 3]
rest = values[1:]
```
